## IDENTITY & ATTRIBUTION

- **Never mention Claude, Anthropic, or any AI tool** in commit messages, code comments, docstrings, markdown files, or any file in this repository.
- **No `Co-Authored-By` lines** in any commit message. Commits are authored solely by Harshit Tiwari.
- **No AI-generated disclaimers** anywhere — no "This was generated by...", no "AI-assisted", nothing.
- If a file already has an attribution comment, remove it when editing that file.

## COMMIT FORMAT

```
type: short imperative summary (under 72 chars)

- bullet: what changed and why (one line per significant change)
- bullet: what file was affected and what it now does differently
```

Types: `feat` (new module/feature), `fix` (bug), `docs` (guide/context update), `style` (frontend), `refactor`.

**Never include:**
- `Co-Authored-By:` lines
- "Generated by", "Claude", "Anthropic", "AI" in any commit field
- Vague summaries like "update files" or "fix stuff"

**Always include:**
- What specifically changed (file name + what it does now)
- Why it changed if non-obvious

## PROJECT RULES (summary — full version in context.md)

- No `matplotlib.show()` anywhere — all output must be terminal-printable
- Python runtime is `python3.14` — no `python` alias; PyTorch has no 3.14 wheel (graceful skip required)
- Every script ends with `if __name__ == "__main__": main()`
- Every script has a `section()` helper and uses `np.random.default_rng(42)` for reproducibility
- Never commit `.env`, `*.pkl`, `*.pt`, `*.pth` files
- Never use `git add -A` or `git add .` — add specific files only
- Never `--amend`, never `--no-verify`
- Confirm with user before pushing to remote (it affects the live site at harshittiwari.me/guide)

## WRITING STYLE (for all .md files)

- Zero fluff — no "In this section we will...", no filler
- Math-first: derive formula, then show code equivalent
- Tables over prose for comparisons
- Dense and technical — every sentence carries information
- LaTeX: `$\mathbf{w}$` (vectors), `$\mathcal{L}$` (loss), `$\mathbb{R}$` (reals), `$\mathcal{N}$` (Gaussian)
