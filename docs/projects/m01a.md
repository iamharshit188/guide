# PCA Image Compressor — Step-by-Step Guide

> **Module:** 01 — Math for ML · **Difficulty:** Beginner · **Type:** GUIDED · **Time:** ~2 hours

## What You'll Build

A CLI tool that loads any grayscale image as a NumPy `float64` matrix, performs PCA using raw eigendecomposition (no sklearn), reconstructs the image at ranks [5, 10, 20, 50], and prints a formatted table showing RMSE, explained variance ratio, and compression ratio for each rank. The core insight: images compress well because pixel variance concentrates in a small number of eigenvectors.

## The Mental Model

PCA finds the directions of maximum variance in your data. For an image treated as a matrix of row-vectors (each row = one image row of pixel intensities), the eigenvectors of the covariance matrix are the "principal pixel patterns." Reconstructing with only the top-$k$ eigenvectors discards low-variance directions — which in natural images correspond to fine noise and texture, not structure. The reconstruction formula $\hat{X} = X_c V_k V_k^T + \bar{X}$ is a low-rank approximation of the centered matrix. Each additional eigenvector adds one rank-1 outer product to the approximation; RMSE drops fast at first, then slowly as you add more ranks — this is the "elbow" in scree plots.

The math in full:

$$X_c = X - \bar{X}, \quad C = \frac{1}{n-1}X_c^T X_c, \quad C v_i = \lambda_i v_i$$

$$\hat{X}_k = X_c V_k V_k^T + \bar{X}, \quad \text{EVR}_k = \frac{\sum_{i=1}^{k}\lambda_i}{\sum_{i=1}^{n}\lambda_i}$$

$$\text{RMSE} = \sqrt{\frac{1}{nm}\|\hat{X}_k - X\|_F^2}$$

## Phase 1 — Load Image and Center Data

**Why this first:** All PCA math operates on the centered matrix. Loading and centering is the data contract every later phase depends on.

```python
#!/usr/bin/env python3
"""PCA image compressor using eigendecomposition only — no sklearn."""

from __future__ import annotations
import sys
import numpy as np
from pathlib import Path
from PIL import Image


def section(title: str) -> None:
    width = 60
    print(f"\n{'=' * width}")
    print(f"  {title}")
    print(f"{'=' * width}")


def load_grayscale(path: str) -> np.ndarray:
    """Load image as float64 matrix in [0, 1]. Shape: (H, W)."""
    img = Image.open(path).convert("L")      # force grayscale
    arr = np.array(img, dtype=np.float64)    # shape (H, W), values 0-255
    return arr / 255.0                        # normalize to [0, 1]


def center(X: np.ndarray) -> tuple[np.ndarray, np.ndarray]:
    """Subtract column means. Returns (X_centered, mean_vec)."""
    mean = X.mean(axis=0)       # shape (W,) — one mean per column (pixel position)
    return X - mean, mean


def main() -> None:
    if len(sys.argv) < 2:
        print("Usage: python pca_compressor.py <image_path>")
        sys.exit(1)

    image_path = sys.argv[1]
    if not Path(image_path).exists():
        print(f"File not found: {image_path}")
        sys.exit(1)

    section("Phase 1 — Load & Center")

    X = load_grayscale(image_path)
    H, W = X.shape
    print(f"Image loaded:  {image_path}")
    print(f"Dimensions:    {H} x {W}  ({H * W:,} pixels)")
    print(f"Dtype:         {X.dtype}")
    print(f"Value range:   [{X.min():.4f}, {X.max():.4f}]")

    X_c, mean_vec = center(X)
    print(f"\nCentered matrix X_c:  shape {X_c.shape}")
    print(f"Column means (mean_vec): shape {mean_vec.shape}")
    print(f"X_c column mean (should ≈ 0): {X_c.mean(axis=0).mean():.2e}")
```

**Checkpoint:** Running with any `.jpg` or `.png` prints image shape without errors. `X_c.mean(axis=0).mean()` is less than `1e-12`.

## Phase 2 — Covariance Matrix and Eigendecomposition

**Why this second:** The covariance matrix encodes relationships between all pixel positions. Its eigenvectors are the principal components. We use `np.linalg.eigh` (not `eig`) because $C$ is symmetric positive semi-definite — `eigh` exploits symmetry for speed and guarantees real eigenvalues in ascending order.

```python
def compute_pca(X_c: np.ndarray) -> tuple[np.ndarray, np.ndarray]:
    """
    Compute covariance matrix and eigendecomposition.
    
    Returns:
        eigenvalues:  shape (W,), ascending order
        eigenvectors: shape (W, W), columns are eigenvectors
    """
    n, d = X_c.shape   # n = H (rows = observations), d = W (features = pixel columns)
    
    # Covariance matrix: (d x d), symmetric
    # Using (n-1) denominator — Bessel's correction for unbiased estimate
    C = (X_c.T @ X_c) / (n - 1)      # shape (W, W)
    
    # eigh returns eigenvalues in ASCENDING order
    # eigenvectors are columns of V
    eigenvalues, eigenvectors = np.linalg.eigh(C)
    
    # Reverse to descending (largest variance first)
    eigenvalues = eigenvalues[::-1].copy()
    eigenvectors = eigenvectors[:, ::-1].copy()
    
    return eigenvalues, eigenvectors


# Add to main(), after Phase 1:
def phase2_demo(X_c: np.ndarray) -> tuple[np.ndarray, np.ndarray]:
    section("Phase 2 — Covariance & Eigendecomposition")
    
    n, d = X_c.shape
    print(f"Covariance matrix will be: ({d} x {d})")
    print(f"Running np.linalg.eigh on ({d} x {d}) matrix...")
    
    eigenvalues, eigenvectors = compute_pca(X_c)
    
    total_variance = eigenvalues.sum()
    print(f"\nTotal variance (sum of eigenvalues): {total_variance:.4f}")
    print(f"Top-5 eigenvalues: {eigenvalues[:5]}")
    print(f"Top-5 EVR: {eigenvalues[:5] / total_variance}")
    
    # Sanity check: eigenvectors should be orthonormal
    VtV = eigenvectors.T @ eigenvectors
    identity_error = np.abs(VtV - np.eye(d)).max()
    print(f"\nOrthonormality check — max |V^T V - I|: {identity_error:.2e}  (expect < 1e-12)")
    
    return eigenvalues, eigenvectors
```

**Checkpoint:** `eigenvalues[0] > eigenvalues[1] > ... > eigenvalues[-1] >= 0`. Orthonormality error is below `1e-12`. For a 256x256 image the covariance matrix is `(256, 256)` — not `(65536, 65536)` — this is the "economy" form treating rows as observations.

## Phase 3 — Project, Reconstruct, and Measure

**Why this third:** The projection onto $k$ eigenvectors creates the compressed representation; reconstruction inverts it. RMSE and EVR together measure quality vs. compression.

```python
def reconstruct_at_rank(
    X_c: np.ndarray,
    mean_vec: np.ndarray,
    eigenvectors: np.ndarray,
    k: int
) -> np.ndarray:
    """
    Reconstruct image using top-k principal components.
    
    Projection:    Z_k = X_c @ V_k          shape (H, k)
    Reconstruction: X̂ = Z_k @ V_k^T + mean  shape (H, W)
    """
    V_k = eigenvectors[:, :k]           # (W, k) — top-k eigenvectors as columns
    Z_k = X_c @ V_k                    # (H, k) — scores (compressed representation)
    X_hat = Z_k @ V_k.T + mean_vec     # (H, W) — reconstructed image
    
    # Clip to valid pixel range
    X_hat = np.clip(X_hat, 0.0, 1.0)
    return X_hat


def rmse(original: np.ndarray, reconstructed: np.ndarray) -> float:
    """Root mean squared error over all pixels."""
    diff = original - reconstructed
    return float(np.sqrt((diff ** 2).mean()))


def explained_variance_ratio(eigenvalues: np.ndarray, k: int) -> float:
    """Fraction of total variance captured by top-k components."""
    return float(eigenvalues[:k].sum() / eigenvalues.sum())


def compression_ratio(H: int, W: int, k: int) -> float:
    """
    Original storage: H * W floats
    Compressed storage: k * (H + W + 1) floats
        - Z_k scores: H * k
        - V_k eigenvectors: W * k
        - mean_vec: W
    Return original / compressed.
    """
    original = H * W
    compressed = k * H + k * W + W
    return original / compressed
```

**Checkpoint:** `reconstruct_at_rank(X_c, mean_vec, eigenvectors, W)` gives back exactly `X` (RMSE near `1e-14`). `reconstruct_at_rank(..., 1)` gives a blurry, barely-recognizable image. `explained_variance_ratio(eigenvalues, 50)` is above 0.90 for most natural images.

## Phase 4 — Print Results Table

**Why this last:** The table makes all measurements comparable at a glance and is the deliverable of this project.

```python
def print_results_table(
    X: np.ndarray,
    X_c: np.ndarray,
    mean_vec: np.ndarray,
    eigenvalues: np.ndarray,
    eigenvectors: np.ndarray,
    ranks: list[int],
) -> None:
    section("Phase 4 — Results Table")
    
    H, W = X.shape
    
    header = f"{'Rank':>6} | {'EVR':>8} | {'Cumul EVR':>10} | {'RMSE':>8} | {'Compress':>10}"
    separator = "-" * len(header)
    
    print(header)
    print(separator)
    
    for k in ranks:
        if k > W:
            print(f"  Rank {k} skipped — exceeds image width ({W})")
            continue
        
        X_hat = reconstruct_at_rank(X_c, mean_vec, eigenvectors, k)
        err   = rmse(X, X_hat)
        evr   = explained_variance_ratio(eigenvalues, k)
        ratio = compression_ratio(H, W, k)
        
        print(f"{k:>6} | {evr:>8.4f} | {evr:>10.4f} | {err:>8.5f} | {ratio:>9.2f}x")
    
    print(separator)
    print(f"\nBaseline (no compression): RMSE = 0.00000, EVR = 1.0000, Ratio = 1.00x")
    print(f"Image dimensions: {H} rows × {W} cols")
    print(f"Full rank: {min(H, W)}")


# ── Complete main() ────────────────────────────────────────────────────────────

def main() -> None:
    if len(sys.argv) < 2:
        print("Usage: python pca_compressor.py <image_path>")
        sys.exit(1)
    
    image_path = sys.argv[1]
    if not Path(image_path).exists():
        print(f"File not found: {image_path}")
        sys.exit(1)
    
    ranks = [5, 10, 20, 50]
    
    # Phase 1
    section("Phase 1 — Load & Center")
    X = load_grayscale(image_path)
    H, W = X.shape
    print(f"Image:      {image_path}  ({H} x {W})")
    print(f"Dtype:      {X.dtype},  range [{X.min():.4f}, {X.max():.4f}]")
    X_c, mean_vec = center(X)
    print(f"Column mean after centering: {X_c.mean(axis=0).mean():.2e}")
    
    # Phase 2
    section("Phase 2 — Covariance & Eigendecomposition")
    print(f"Covariance matrix: ({W} x {W})")
    eigenvalues, eigenvectors = compute_pca(X_c)
    total_var = eigenvalues.sum()
    print(f"Total variance: {total_var:.4f}")
    print(f"Top-3 eigenvalues: {eigenvalues[:3].round(4)}")
    VtV_err = np.abs(eigenvectors.T @ eigenvectors - np.eye(W)).max()
    print(f"Orthonormality error: {VtV_err:.2e}")
    
    # Phase 3 + 4
    print_results_table(X, X_c, mean_vec, eigenvalues, eigenvectors, ranks)


if __name__ == "__main__":
    main()
```

**Checkpoint:** Output table has 4 rows (or fewer if image is narrow). RMSE at rank 50 is below 0.05 for natural images. Compression ratio at rank 5 is above 10x for a 256x256 image.

## Debugging Guide

| Symptom | Likely Cause | Fix |
|---------|-------------|-----|
| `RMSE` at full rank is not near zero | Centering not applied during reconstruction — `mean_vec` missing in `X_hat` | Verify `X_hat = Z_k @ V_k.T + mean_vec`, not just `Z_k @ V_k.T` |
| Eigenvalues contain small negative values (e.g. `-1e-15`) | Floating-point error in `eigh` — covariance matrix is positive semi-definite, not definite | Clip: `eigenvalues = np.maximum(eigenvalues, 0)` before EVR computation |
| EVR at rank 5 is below 0.3 for a photo | Image was not converted to grayscale — color channels add variance spread | Confirm `Image.open(...).convert("L")` before `np.array` |
| `MemoryError` on large images | Covariance matrix is `(W x W)` — if W is 4000+, that's 128MB of float64 | Resize image first: `img = img.resize((512, 512))` |
| Reconstructed image has values outside [0, 1] | Reconstruction can overshoot at low rank | `np.clip(X_hat, 0.0, 1.0)` before RMSE — already in template |
| `eigenvectors` shape assertion fails | `eigh` returns eigenvectors as rows in some versions — check documentation | Always index as `eigenvectors[:, :k]` (columns), never `eigenvectors[:k, :]` |

## What This Project Teaches You to Say in an Interview

1. "PCA is equivalent to truncated SVD on the centered data matrix — both find the same subspace; eigendecomposition of the covariance is cheaper when features outnumber samples."
2. "Reconstruction error is bounded below by the sum of the dropped eigenvalues: $\|X_c - \hat{X}_c\|_F^2 = \sum_{i=k+1}^{n} \lambda_i$, so you can predict RMSE before computing it."
3. "We use `np.linalg.eigh` over `eig` because the covariance matrix is symmetric — `eigh` is $O(n^3/3)$ instead of $O(n^3)$ and guarantees real eigenvalues without complex arithmetic."
4. "Compression ratio isn't just $n/k$ — it's the ratio of original storage to the storage cost of scores plus eigenvectors: $HW / (k(H+W+1))$. For small $k$ this is much larger than $n/k$."
5. "The independence assumption in 'naive' compression is that pixel columns are independent given the principal components — PCA relaxes this by finding correlations, which is why images compress well: adjacent pixels are highly correlated."
